How to Download Backup of Website From cPanel

We have discused earlier the complete process of creating website backup in the post `How to create backup of my website` now I’m going to teach you how to download that created backup to your hard disk or any storage device from cPanel. Many people struggle a lot to find and download blog’s backup file from cPanel even after successfully creating the Backup, they find it very complex to reach the place where backup is stored in cpanel or the place where they can get a downloadable file of blog’s backup. I’m going to explain the complete procedure in detail needed to download a website’s or blog’s  backup from cPanel.

How to Download Blog’s Backup from cPanel to Hard disk..

For downlaoding backup of your WordPress blog or website from cPanel you just need to perform some very easy steps and it will take only few minutes, here we go.. 1. Log in to cPanel. 2. Click on File Manager in file section. 3. Select Home Directory and then Go. 4. Now you will find a lot files there, just scan the list to find backup file (it will be there with the name like `backup- 1.3.2013 _17-38-o1_harkuchh.tar.gz`) just click on it. 5. Finally on the top of the screen click on download and it will start immediately and take time depend on the size of backup file and speed of Internet obviously. Believe me it’s the easiest way to download backup of your blog and website.
