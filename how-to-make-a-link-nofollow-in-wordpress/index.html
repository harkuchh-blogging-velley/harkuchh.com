How to Make a Link Nofollow in WordPress

Before proceeding, I think knowing the difference between a no-follow and a do-follow link is very vital, Nofollow and dofollow are two very oftenly used words in SEO. Blogging became very popular in last half decade, but many Bloggers of our generation making a big mistake in blogging, and it is spreading of dofollow links to other website-blogs unnecessarily. See, while writing post on your blog when you give a link of any other blog somewhere in your post it becomes a dofollow link and by this way you are giving a free do follow link to that blog, which is really not needed. A normal Dofollow link allows all search engines to follow them and reach the linked website. It give link juice and a back link. If a webmaster is linking back to you with this link both Search Engine and people will be able to follow. Enable The best way to give someone dofollow love making keyword in the anchor text. This means that when you use a link to a website or page, the targeted keywords as anchor text. A modified Nofollow links does not allow any search engine robot to surf the link, now here point to pay attention is that, if you’ll give a nofollow attribute tag to a links on your web page or website it links you from other page but because of nofollow attribute, it does not pass link juice. Any Reader can only follow the links but search engines don’t. It’s a strong Google’s recommendation that when we link our article with any other website we must have to be very sure that it’s a good website. Sometimes we need to link some p0rn , hacking kind of sites which are generally not consider as a legit or good site, there you will have to add nofollow link attribute. It becomes even more necessary if you are using affiliate marketing, and keep adding affiliate links on your blog. Being truthful, adding a nofollow tag to any link is damn easy, in order to make any link nofollow, all you need to do is `add nofollow attributes in HTML formal` for example <a href=”http://royalyouth.com” rel=”nofollow” > Royalyouth</a> But I’m pretty sure many of us wants to get things done more easier, I know some might not like the idea of using HTML to add Nofollow link attribute.
