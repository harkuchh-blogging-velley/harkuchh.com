Why I Am Learning Python

Feb 11th, 2014

Why I’m learning Python? well yes, this is really a very valid question and it has a solid answer too, I think everybody should read this, not because I’ve written this for you but because it has much much importance and advantages for you.

Some days back, I was in my office with my friend Channi (Charanjit Singh) we were planning to work on a website for a special purpose. While talking I said him that, “on this website we’ll need to do same thing again and again hundreds of time and it will take lots of our time and effort :(“. Then he said “Don’t worry dude, I’ll make a script that will automate all this thing and then you will need to do that only once and rest will be done by the script.”

I was surprised and like, is it possible? He said “Yeah dude, it’s simple and will not take too long to be developed” then I told him to do it fast, and I started writing a post on my movie blog and when I published only one post He said “Done bro! here it is.”

I said “seriously, will it do all those things which I was talking about?” I got the best reply, it was ‘off course!’.

I said let’s see, and unbelievably it done all those things in just minutes, any person who have no knowledge of coding and programming will never believe that the things which required at least 1 week day and night work was done in just minutes. I was shocked! and said to my friend, “Awesome bro, how did you do it”, he replied “Python bro, it’s awesome”.

Till then I just knew that Python is a programming language but I didn’t know anything about its working or coding. Actually I didn’t know any coding or programming language. Not even C++ in which I spent all four years of my B.tech degree.

So when I heard Python, I didn’t take even a second to say, “bro! can’t I learn Python?”

“Of course you can”, he said. “But you will need to be regular on it”, he added.

I replied joyfully “of course I will, it will be great if I can do things in minute(s) for which I give hours, days and even weeks to complete, so let’s start it now.”

He said No! We will start it from tommorow, let me prepare modules and a structure for teaching Python in the best way to you. And next day we started.

And now, just after a week.. Even I cannot believe that now I can do many things with my own created crawlers which none of you can do even in weeks, please note, I’m not showing off here but I’m just trying to tell, how much difference simple coding are able to create.

Let me share an example with you so that you can get my words better, previously I used to copy contents of events websites manually in which I needed to..

  • Go to a website

  • Open a post

  • Create a new text file (to copy post)

  • Copy posts title from that website

  • Paste that into text file

  • Copy body of the post

  • Paste that into text file again

  • Copy keywords, description and tags from source code

  • Paste them into text file

…and these all steps were required for every post and imagine if there were 100 posts on that site it simply means I needed to do these all things 100 times to get everything in my hand, most of you are doing same, I know. But..

today, I just take some minutes to create a crawler for any specific website and run it, after that crawler do all the rest thing in minutes. Any non technical person will never believe that, it copies posts at the rate of 500-600/minute with proper distinguish between every part of the post which includes title, body, images, keywords, tags, description, category and all related information which I think I require and I told, to to do same thing manually I required at least 2-3 days.

So this is the difference and that all is possible with a simple effort to learn python. That is why I’m learning Python.
