How to Add Facebook Like Box to My Website

When a bloggers start blogging, the first thing he need to do is the promotion of his blog  through which he can get initial traffic to his blogs and Facebook Like Box is the Primary tool to start Advertisement of any blog. Many time WordPress Themes have predefined option to add Facebook like box in WordPress through Widgets, but sometimes it’s not given in themes then the blogger manually has to add the FB Like Box to the blog as this is really important.

How to add Facebook like box to Website

Adding Facebook like box to WordPress website or blog is really very easy you just need a Facebook Fan page, if you have already a page then you won’t need to create, else you have to create a Facebook fan page first, then adding that page to blog requires few simple steps to do and it won’t take more than 5 minutes, just follow the steps given below..  
  1. Log in to your Facebook Account.
  2. Open your site’s Fan page and copy it’s URL (Complete Address)
  3. Now click on Edit Page > Update Info in Admin Panel at top.
  4. Click on Resources given at left sidebar.
  5. Click on Use social Plugins.
  6. Go down and click on Like Box.
  7. Paste your Page URL in box front of you and edit the settins for box size, faces etc.
  8. >Now just click on Get Code Button and then IFRAME.
  9. Copy the Text Code shown in the Box.
  10. Go to your WordPress account and open Widgets.
  11. Click on Text box and drag it into the Widgets.
  12. Paste Code in Text Box and click on save.
You Done ! check out your blog. :) If you don’t want to do that all technical stuff and want to make it simple and short, copy the code given below and paste it inside your WordPress account where you want to place fb like box. Change the colored URL with your own Facebook page and you’re done.
<style type=”text/css”> .facebookOuter { background-color:#F4F4F4; width:300px; padding:10px 0 10px 10px; height:145px; border:1px solid #CCCCCC; } .facebookInner { height:159px; overflow:hidden; } </style> <div> <div> <div data-width=”300” data-height=”180” data-href=”https://www.facebook.com/Harkuchhcom” data-border-color=”#808080” data-show-faces=”true” data-stream=”false” data-header=”false”> </div> </div> </div> <div id=”fb-root”></div> <script>(function(d, s, id) { var js, fjs = d.getElementsByTagName(s)[0]; if (d.getElementById(id)) return; js = d.createElement(s); js.id = id; js.src = “//connect.facebook.net/en_US/all.js#xfbml=1”; fjs.parentNode.insertBefore(js, fjs); }(document, ‘script’, ‘facebook-jssdk’)); </script>
    You can change the background color of Facebook like box here default color for background is #808080, and even if you want to modify the size of Like box, you can easily change them by changing values of height and width defined in lines 4, 6, 10 and 17, Over !
