Largest Memory Unit, Detailed List : Smallest to Biggest

In today’s world of technology we use memory and it’s terms everywhere, we know memory is the physical devices used to store all kind of digital programs, and data for temporary or permanent basis for access those data in computer or other digital electronic devices. KB, MB, GB are terms which are used everywhere to distinguish data size, some of us know that data unit starts from a bit which is equal to a binary character but have you ever wondered what is the maximum memory unit, I know you don’t ! and even if you have wondered of it, then let me tell you, whatever you have wondered is too less.

Here is the detailed list of digital memory units smallest to biggest See this and think How far do u know.

1 Bit : 1 Binary Digit
4 bit : 1 Nibble
8 Bit : 1 Byte
1024 Byte : 1 Kilobyte (KB)
1024 Kilobyte : 1 Megabyte (MB)
1024 Megabyte : 1 Gigabyte (GB)
1024 Gigabyte : 1 Terabyte (TB)
1024 Terabyte : 1 Petabyte (PB)
1024 Petabyte : 1 Exabyte (EB)
1024 Exabyte : 1 Zettabyte (ZB)
1024 Zettabyte : 1 Yottabyte (YB)
——————————————— 10
1024 Yottabyte : 1 Brontobyte
1024 Brontobyte : 1 Geopbyte
1024 Geopbyte : 1 Saganbyte
1024 Saganbyte : 1 Pijabyte
1024 Pijabyte : 1 Alphabyte
1024 Alphabyte : 1 Kryatbyte
1024 Kryatbyte : 1 Amosbyte
1024 Amosbyte : 1 Pectrolbyte
1024 Pectrolbyte : 1 Bolgerbyte
1024 Bolgerbyte : 1 Sambobyte
——————————————— 20
1024 Sambobyte : 1 Quesabyte
1024 Quesabyte : 1 Kinsabyte
1024 Kinsabyte : 1 Rutherbyte
1024 Rutherbyte : 1 Dubnibyte
1024 Dubnibyte : 1 Seaborgbyte
1024 Seaborgbyte : 1 Bohrbyte
1024 Bohrbyte : 1 Hassiubyte
1024 Hassiubyte : 1 Meitnerbyte
1024 Meitnerbyte : 1 Darmstadbyte
1024 Darmstadbyte : 1 Roentbyte
———————————————- 30
1024 Roentbyte : 1 Coperbyte
1024 Coperbyte : 1 Koentekbyte
1024 Koentekbyte : 1 Silvanikbyte
1024 Silvanikbyte : 1 Golvanikbyte
1024 Golvanikbyte : 1 Platvanikbyte
1024 Platvanikbyte : 1 Einstanikbyte
1024 Einstanikbyte : 1 Emeranikbyte
1024 Emeranikbyte : 1 Rubanikbyte
1024 Rubanikbyte : 1 Diamonikbyte
1024 Diamonikbyte : 1 Amazonikbyte
———————————————– 40
1024 Amazonikbyte : 1 Nilevanikbyte
1024 Nilevanikbyte : 1 Infinitybyte
1024 Infinitybyte : 1 Websitebyte
Note : There is no official measurement for memory unit size after Yottabyte yet, but these are suggestions with strong potential, so from now we can say websitebyte is the largest memory unit for digital devices.
