Tricks to Search Amazing Things With Google

Feb 17th, 2014

This gonna be a long post, but I bet you won’t want to miss a single word. Krishan tried to convince me to break it into 4 different posts by giving multiple references from all his blogging ‘knowledge’ (here’s a pun if you can get it), but I din’t agree. Quality of content is more important imo.

Here I am telling best of the tricks I use to get what I want from Google. They are fun. If you have some more you think I should know, please mention in comments. Enjoy!

Exclude all results of a website

So many times we need to search something but results from some spammy website clutter all search results. Search with following query to remove results from a particular website

-site:“spamy_site_name.com”

e.g

big bang theory episode -site:youtube.com

Above query will remove all results of youtube.com.

How it work

You know already if would have read my last post about Google Syntax. There are two things working here.

  • “–”
    The minus(‘–’) sign tells google to exclude results of the query directly after it.
  • site:
    This tells google the website to be excluded

Direct download anything

We all love downloading free music, right? But discovering music has become a tedious task with all those spam sites punching ads in our faces.
Here’s a trick to get to direct links of songs on a website. Search for the following query and replace desired fields to get direct links (most of the time):

intitle:“index of” mp3 wadali brothers

You can replace anything after and including ‘mp3’.
In my school days I created a bot to download porn using this technique. I had one hell of a porn collection back then. I still curse me for deleting it :–/

How it work

This trick uses the fact that people on Internet are not actually all smart. There are pretty stupid people around here.

What we are actually doing here is that we are not searching for any pages created by site developer, but for the default pages which server generates for filesystem access over Http. Don’t worry if you didn’t get it.

Try opening something like “file:///”. There is no web page over there, but your browser might be showing you a list of files. This is exactly what we are searching for, a page with direct access to the folder on server where ‘mp3’ are saved.

Beware of spam

I am not the only one who know this trick. Spammers know it very well too. This query will work without a glitch for uncommon terms (like ‘wadali brothers’) which are not very popular. But if you search for something remotely famous (or too general like hindi songs), you will find herds of spam sites rolling on your site.

Detect spam

How can you differentiate b/w spam page and real page you are looking for?
Well, as I said, we are not actually looking for any page at all. So if you see anything stylish (like colors, styles, ads, sidebars), it’s not a real page. Real thing we are looking for do not have anything but white background, black text, blue links (violet visited links), a header and an untitled list. Nothing more.

How to Avoid spam

  • Use the previous technique of excluding all results from a website
    Just keep adding -site:spamsite.com for every spam site you don’t want to see.

  • Avoid too general queries
    Try to search for something specific like the song name or singer name. It works to a limit though. If you are searching for something very popular, spam is there waiting for you.

Download books and documents

This one is my favorite. I read a lot of books. So of course I download a lot of them. This technique can actually be used to download any kind of document available in some specific file extension.

ext:pdf how to think like computer scientist python

And there you get direct links to the pdf file. You can replace ‘pdf’ with ‘doc’,‘odf’,‘epub’ or any other format. You will likely meet less spam in this dimension, but there are times when you’ll not find a book. Publicly available free books will generally be there though.

How it work

It search for files with specific file extension. Duh! How else it might be working?

View public cameras

You can search cameras which have public access. I mean you can get access to the video stream of some publicly available video cameras. Sometimes there are controls available as well for controlling cameras.

You can look through cameras placed in many places like traffic lights, shopping stores, offices and even home cameras sometime. You need to search for special queries for this. They are kinda hard to remember, so even I’ve made a list of those queries.

How it work

It’s same technique as ‘direct download anything’. We are searching for some ‘special’ pages on the net that Google has (by mistake?) crawled.
