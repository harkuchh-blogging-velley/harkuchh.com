Trick to Copy Content From a Website With Copy Protector Enabled

Jan 29th, 2014

Copy protector is a wordpress plugin which is used by many wordpress sites to stop readers from copying the site content. When you visit a site and find it good to share some part of it, this type of plugins irritate us the most. In these cases, you either share the whole page, or you don’t share anything.

There are many such plugins which prevent users from copying partial content from a website. Here we have a solution to this problem.

The solution we are mentioning here work against all plugins which prevent copying content. This means you don’t need to know which plugin is used by the website that is stopping you from copying, following solution simply work against anything.

How do these plugins work?

Although this is not necessary to stop those copy protector plugins, but having proper knowledge is a good thing.

What we see on a web page is HTML content which is styled with CSS stylesheets. HTML and CSS do not offer much programming power to do anything like stopping people from copying content.
But there is a third power on the webpage which is actually used for programming fancy things on a web page. Its called javascript.

Javascript is a web scripting language that run in our browser, which can do many many awesome things. Preventing us from copying is one of them. So if we can somehow disable the execution of javascript in our browser, all those plugins which stop us from copying content will simply stop working.

But how do we do it?

In previous versions of most browsers, one setting was provided to switch off javascript. But not anymore. New browser no more provide this option since more and more websites are becoming javascript heavy and they simply do not work if javascript is disabled.
So we need a third party tool which can disable javascript in the browser.

Browser plugins to the rescue

Most browsers have addons or plugins which we can install in the browser. Various plugins are available to disable javascript in browsers. We can search the browser addon market and find many.

Our favorite plugin for this purpose

No Script

The NoScript Firefox extension provides extra protection for Firefox browsers. It is free and open source add-on which allows JavaScript, Java, Flash and other plugins to be executed only by trusted web sites of your choice. NoScript also provides the most powerful anti-XSS and anti-Clickjacking protection ever available in a browser.
With noscript we can choose which site can execute javascript and which can’t.

So what we actually have to do?

Just install noscript in firefox and follow the tutorial.

Tutorial to copy content from a website with Copy Protector enabled

Okay this is a really small tutorial here.
When you have noscript installed in firefox, there will appear an icon like this in your firefox window.

By default noscript do not allow any site to execute any javascript, java or flash code. So most sites will just stop working the way they usually do.

So first thing we have to do is disable the noscript gloablly.

  • Click on the noscript icon and unselect “Forbid Scripts globally”.

It will show a warning messag saying

“Scripts are going to be allowed globally (for every site). This is a potentially dangerous action. Do you really want to proceed?”

Just ignore it and press ok.

  • Now open the site from which you want to copy content.
  • With that site’s page open, again click on the no-script icon and select “Mark <site-name> as Untrusted”

Doing this will auto-refresh the page and no-scripts will be run from that page. This means none of the copy blocking plugins will work.

So, that’s what we wanted to achieve. If you have any questions and queries, please feel free to ask in comments. We also have forums where you can ask any question and me and our team will try to answer as per our knowledge.
