How to Change Permalink Settings in Wordpress

What is permalink

WordPress is Website making Software, In WordPress when we place any post to our website or blog WordPress uses default web URL’s for the new posts which is very complex because it having many question marks, numbers and symbols, However WordPress offers us the ability to create a custom URL structure for our permalinks and archives. This will improve the aesthetics, usability, and forward-compatibility of your links and it will be easily optimize by the search engines.

Example :

Default URL : http://www.google.co.in/#hl=en&sclient=psy-ab&q=tuneup+utilities+2012+product+key%5C&oq=tuneup+utilities+2012+prod

Changed URL :

http://harkuchh.com/how-to-access-multiple-facebook-accounts-in-same-browser/  

Now the steps you have to follow for change permalink are given below :

  1. go to posts
  2. Click on Add New Post
  3. Write the title of your post
  4. Just below it, you will find “permalink”
  5. Click on edit/Settings
  6. you will get a lot of options , SELECT ” Sample-post”
  7. click on Save settings
You have done, This is as simple as that :)

You Might also Do this

  1.   Click on the Settings tab in the WordPress dashboard
  2.   Click on Permalink
  3.  you will get a lot of options , SELECT ” Sample-post”
click on Save settings, Again you have done enjoy :) :)   Now please share this info on face and twitter to let all other Blogger know this great information…:)
