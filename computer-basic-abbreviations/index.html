Computer Basic Abbreviations..

Computer is the most necessary electronic machine for every individual, Now almost everything is available on Computer and internet now it is impossible to imagine our life without internet, Some computer basic  abbreviations are placed here for you people which are normally used where computer system is being use… COM : Computer PC     : Personal Computer USB  : Universal serial Bus CPU  : Central Processing Unit APP  : Application NET  : Internet MAIL : e-mail INFO  : Information REQ    : Request ACK    : Acknowledgement PSW    : Password WWW : World Wide Web SITE  : Website FB       : Facebook HDD   : Hard Disk Drive FDD    : Floppy Disk Drive LCD    : Liquid Crystal Display RAM  : Random Access Memory ROM  : Read Only Memory SATA : Serial Advanced Technology Attachment IC      : Integrated Circuit MP   : Microprocessor RD    : Read WR   : Write REG : Register CLK : Clock I/O : Input Output IP   : Internet Protocol LAN    : Local Area Network MAN : Manupulated Area Network WAN : Wide Area Network MUX : Multiplexer URL : Unoform Resource Locator WAP : Wireless Access Point PRGM : Program SEO   : Search Engine Optimization OSI   : Open System Interconnection ISO   : International Standard Organisation UDP : User Datagram Protocol FTP  : File Transfer Protocol USART : Universal Asynchronous Receiver/Transmitter ADMIN : Administrator CTRL : Control SHFT : Shift If you found it useful must share this..
