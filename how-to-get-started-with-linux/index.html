How to Get Started With Linux

So are you eventually tired of Windows’ crashes and viruses ? I know many who are. It’s simply not worth to keep using a system which can’t even function properly (Windows you hearing me ?). I know you have heard of that Linux thing but never tried. Yes the rumors are true,Linux never crashes, and it never hangs. You don’t even need to press the refresh again and again to keep it working properly. It’s awesomeness in its most raw form. If you are reading this post you are most likely willing to get out of the jail Microsoft has put you in (and they even ask you to pay for it), or you are standing on the line to make the decision to give Linux a shot. If it’s the latter case, here are some features of Linux which should give you the final push.

Stable to its core

As I stated earlier, it’s a solid system. So solid that sometimes I am forced to wonder that even Ambuja Cement is no match for it. You can leave a computer running for a month in a cyber cafe, and even after the month it will be working as fast as ever, without any need of restart or anything at all. Hell, most servers run Linux (that include Google, Yahoo, Facebook and more) and they run literally for years non stop. Try that with your Windows machine ^_^

No viruses ever (god promise)

I can bet on this. Being a power user (and a proud geek) for about three years, I never ever faced a virus in Linux (never even heard of any). This is one thing you gonna miss the most once you start using (or enjoying I would say) Linux. How come Linux has no viruses? Read our previous post on Why virus don’t work on Linux.

No Licenses

Don’t want to pay for your “WINDOWS” anymore? You don’t need to. Linux is all free. You don’t need to pay a single paisa for it, and nobody will ask you to do that. You no more need to live in the fear that if you mistakenly update your windows someday, Microsoft will block your computer. No money? No worries, Linux is all free and open source (if you care about that).

Free Free Free

Almost every software on Linux is free too. Literally each and everyone of these and millions of more.
  • Free Office suite
  • Free Photoshop, Gimp we call it
  • Free Animation Studio
  • Free DJ Studio

Linux is hard to use ?

I heard someone saying that Linux is very hard to use. May be it was in 90’s. But. Wake up Sid, Geek is the new sexy anyway. Although you don’t need to be a geek to use Linux, it’s too sexy man. Remember what Steve Jobs said? You need to think different. Very well then, Follow my lead.

First steps in animal territory

Taste Linux without loosing Windows safe-house (pun intended)

Yup! You can install Linux like a windows program and then if you don’t like it just uninstall it like anything else. It will still be the full blown Linux Operating System, but it will not affect your Windows. No, I am not talking about Virtual Box or VMware. What we are going to do is use a software which will install the Ububtu (most popular Linux) like any windows application.

Here’s how to do the magic

It’s basically the Ubuntu (very famous Linux distribution) which is installed with a program (WUBI) provided by Canonical (people that make Ubuntu)
  • Download the latest Ubuntu iso file
  • Download latest Ubuntu Windows Installer (wubi we call it)
  • Place the .iso file and wubi.exe in same folder and run wubi.exe
  • Then Choose the drive you want to install Ubuntu in
  • Don’t worry, C: will be just fine too. It will not harm your computer in any way.
  • Choose how much space you want to give the new OS (1 GB is enough)
  • Click the magic button (Install I mean)
  • Restart the Computer
  • Then on startup, you will be given two choices like in this screen
  • Make your choice and you are welcome :)

Using Linux

So now we can run Linux easily and play with it like a toy. But what next? Ubuntu has a very straight forward approach, nice looks and slick interface. There is no need of much illustration. Feel free to enjoy it to your heart’s content. Here are some tips and tricks about some new things you gonna find in Ubuntu.

Installing new software

Installing new software in Linux is different from Windows. There are no .exe files or other setups like that which you can use to install software. Yea, that means you no longer need to click those next-next buttons for every software you want to install. Instead there is a software center where you can search for software, read the description about what the software do and install it if you want. The latest version of the software will then get downloaded and installed on your new machine. It’s more like an app store, and much powerful system then Windows’ ‘setup’ approach. We’ll discuss in some other post what’s so amazing about Linux approach of installing software.

Removing software

You guessed it right, the software center. It’s your one stop shop for every new software you want to install or remove.

What if you are offline ?

That’s actually worst than it sounds. There are many problems you’ll face using Linux offline. If you are using a computer which can’t stay online, I recommend not trying Linux on this machine. That is if you are new to Linux world. There are of course solutions to those problems (follow same link given above), but they are kind of hard for Linux beginners.

Watching videos and listening music

You will not be able to listen to music or watch videos on Ubuntu you just installed, lol. Actually you can, but you need to download extra software for that. Why is it like this? Well, it’s because the decoder packages which are used to play music (mp3) and videos are not open source. In order to keep itself completely open and free, Ubuntu do not offer those software in default CD. You need to install them later.

How to do that ?

Simply try to play a mp3 and/or a video and Ubuntu will ask you to download and install some plugins. Click ‘Install’ and get going. It will download about 28-35MB, and here you have the safest machine you can get with little to no effort on your part.

Why should you use Linux ?

If the reasons I gave in beginning of this article were not enough, here are some use cases that may convince you.

To Surf Internet

Which is the safest way to surf the web? There is none. But you can minimize the risk by using Linux. Most bad websites assume you to be using Windows and try to hack using scripts which think exploit several ‘features’(?) of windows. Using Linux means automatic safety from 70% of those bad websites.

Lack of Anti-virus

Are you tired of your antivirus screaming in your face to buy the pro version or singing the songs that its trial version is expired? No virus means no need for anti-virus. Use Linux, the ‘Ram-baan’ for all viruses.

Transfer data from pen drives

If you are like me (and most people I know), there will be lots of times when you will need to use a pen drive (or data card) but you are scared that the pen drive would have virus. Just boot into Ubuntu and use it for all data transfer. Virus don’t spread by itself, it spread only when you open a virus file (which happens automatically when you open a pen drive). Using Linux can save your ass every time you need to use a pen drive. It is not that hard to use Linux as people say. Use this guide to help you, or you can contact me if you have any problem. I will be more than happy to help. Linux is the awesomeness you must not ignore. Even if you need to suffer a little with your first steps towards Linux, it’s worth the fruit you gonna get in the end. Happy hacking :)
