How to Use Google Like a Pro: Tips to Use Google Syntax

Jan 30th, 2014

I know you all use Google to reach everywhere on the Internet, but are you using it properly? Google actually provides a specific syntax for searching which is unknown to the masses. Also known as the Google operators, this syntax can make your Googling much easy and much less frustrating.

It’s specially useful for bloggers since all of us need to keep an eye on the Internet trends. It’s very useful to know how to properly use our most used tool, that is Google.

So let’s waste less time an look at what Google has to offer for us.

How to use Google syntax?

All the search operators described here can be used by putting the operator in the search box followed by the search query. Like to use related: operator, following query will be used

related:"harkuchh.com"

related: Search for related sites

related:"example.com"

related: is used to search for websites which are similar to the website we provide in search box. When we use this operator, Google will search for any information related to the given website from all over the Internet. Google will show websites which have similar content.

link: Search for all pages that link to a particular url

link:"http://harkuchh.com/trick-to-copy-content-from-a-website-with-copy-protector-enabled/"

This operator will search for all the pages that link to the given url.

site: Search within a particular website only

"site:"harkuchh.com" blogging tips

Above query will only search ‘harkuchh.com’ for the words ‘blogging tips’. This is a very useful operator. Just imagine when you need to find a particular post you once seen on a website but can’t find a link to it. It’s one of my most used operators.

Search for an exact phrase or word

When we search for a sentence on Google, it doesn’t just search for that exact sentence. Try searching for

I will become dictator of the world

Google will search for text which have ‘I’, ‘will’, ‘become’ etc anywhere on the page. But if we want to only search for pages which contain the complete sentence “I will become dictator of the world”, we can do it just by putting the whole sentence within quotes. Like this

"I will become dictator of the world"

Exclude something in the results

Sometimes we want that some specific type of results should not appear in the search results. Google provide a special operator just for this purpose.
That is ‘–’.
Yes just that. Putting a ‘–’ in front of any word will show results which do not have that word in them. To make it even better, this work with other Google operators as well.

hello lady how are you -season

Above query if searched without ‘-season’ will show some youtube results about some seasons of some TV program. But running above query will remove those results which have the word ‘season’ in them.
