What Is Subdomain ? How to Create SubDomain of My Website

Few days back I created a Subdomain of my website and when I advertised it on Facebook one of my friend asked me Hey K.K, How did you created Subdomain of your website then I said to him that it’s very easy, you can we can create sub-domain of your website through cPanel. What is Subdomain ?

Subdomains are the third level domains that are used to organize different types of your web site content. Suppose you have a website on Automobile and now you also want to create a site for Gadgets at that time rather than purchasing a new domain, you can create sub domain using the domain you already own, and by this way you can create multiple site by purchasing only one domain. Same like that I wanted to create a website for career tips then I simply made a subdomain (careerguide.harkuchh.com) of my domain name Harkuchh.com.  

How to Create Subdomain of My Website ?

To create a Subdomain just follow instruction given below..
  1. Log in to cPanel.
  2. click on Subdomains under Domain Section.
  3. Type the subdomain name you want in the box before the domain name.
  4. Click inside the ”Document Root” box and it will auto-fill.
  5. Hit the Create button.
  6. You Will see a message that Subdomain has been created ! Now Go back to cPanel Home. 
 

How to Install WordPress in Sub-domain

After making a Subdomain You must need to install a CMS to work in it, so in order to Install Wordpress Instructions are given below..
  1. Click on Fantastico De Luxe in cPanel.
  2. Now Click on WordPress at left.
  3. Carefully choose Domain in which you want to Install WordPress.
  4. Fill Admin Username and Password in Admin access data section.
  5. Click on Create.
  6. Done ! now open your subdomain Wordpress account and work. :lol:

FAQ’s regarding Subdomains

 

Can I delete Subdomain after once created ?

Yes, You can delete is anytime you want from cPanel.

How many Subdomains can I create on one Domain name?

It’s depend on your hosting company, generally 25 Subdomain are allowed with one domain.

Can I Use Different CMS in Subdomains ?

Yes, If you are using Wordpress in main Domain you are free to choose any CMS for subdomain like Wordpress, Jhoomla.
